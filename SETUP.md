# دليل الإعداد - AWA CYBER

## الخطوات الأولى

### 1. تثبيت المتطلبات

```bash
npm install
```

### 2. تشغيل المشروع

```bash
npm run dev
```

ستكون الصفحة الرئيسية متاحة على:
- الإنجليزية: `http://localhost:3000` أو `http://localhost:3000/en`
- العربية: `http://localhost:3000/ar`

## البنية الأساسية

### المكونات المنشأة

#### 1. **Navbar** (`components/Navbar.tsx`)
- شريط تنقل ثابت في الأعلى
- تبديل اللغة (العربية/الإنجليزية)
- روابط التنقل الرئيسية
- زر CTA "Claim Your Free Quotation"

#### 2. **Hero Section** (`components/Hero.tsx`)
- قسم البطل الرئيسي مع خلفية متدرجة
- عنوان رئيسي مع تسليط الضوء على كلمات رئيسية
- زرين CTA
- أسهم التنقل (للكاروسيل المستقبلي)
- مؤشر التمرير للأسفل

### مكونات UI (`components/ui/`)

تم إنشاء مكتبة مكونات قابلة لإعادة الاستخدام:

1. **Button** - أزرار متنوعة الأنماط
2. **Card** - بطاقات لعرض المحتوى
3. **Badge** - شارات للتصنيفات
4. **Input** - حقول إدخال
5. **Textarea** - مناطق نص
6. **Separator** - فواصل
7. **Container** - حاويات بأحجام مختلفة

### نظام الترجمة

- الملفات في `messages/`:
  - `en.json` - الترجمات الإنجليزية
  - `ar.json` - الترجمات العربية
- الدعم التلقائي للـ RTL عند استخدام العربية
- تبديل سلس بين اللغات
